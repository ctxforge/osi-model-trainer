# Модель OSI — Интерактивный тренажёр

Веб-приложение для изучения сетевой модели OSI. Чистый HTML + CSS + JavaScript, без фреймворков и зависимостей.

## Разделы

- **Главная** — обзор модели OSI с интерактивной башней
- **Изучение** — подробная информация о каждом из 7 уровней + сравнение OSI и TCP/IP
- **Симулятор инкапсуляции** — визуализация процесса инкапсуляции/деинкапсуляции
- **Лаборатория** — эксперименты с сетевыми параметрами (передача пакетов, фрагментация MTU, маршрутизация и TTL, IP-калькулятор)
- **Drag & Drop тренажёр** — сопоставление протоколов и устройств с уровнями OSI

## Деплой

GitHub Pages (ветка `main`, корневая папка `/`).
